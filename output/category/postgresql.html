json vs jsonb

Let's assume that we all agree that json and jsonb are both useful. But what is better(faster)? This question will be answered in the following benchmark. Here you can find a python script I used to generate tests. Actually, all of the benchmark tests look like 

select testold('{}'::json)::json;

or

select testnew('{}'::jsonb);

Where "testnew" is used to test jsonb and "testold" is used to test json so that we can clearly define what were the results of jsonb and json. ![benchmark results][transform]
